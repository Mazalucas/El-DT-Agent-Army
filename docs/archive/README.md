# Documentación de Referencia

Este directorio contiene documentación histórica y de referencia que no es esencial para usar el framework, pero es útil para entender el contexto y la evolución del proyecto.

## Contenido

- **INSPIRATION.md** - Fuentes de inspiración (OpenAI Cookbook, CrewAI, etc.)
- **RESEARCH.md** - Investigación inicial sobre arquitecturas de agentes
- **SPECIFICATIONS.md** - Especificaciones v1 (deprecated, ver SPECIFICATIONS_V2.md)
- **PLAN_REVIEW.md** - Revisión completa del plan de implementación
- **CREWAI_LEARNINGS.md** - Lecciones aprendidas de CrewAI
- **TASKMASTER_RULES_INTEGRATION.md** - Integración de reglas de taskmaster
- **DT_AUTONOMY.md** - Detalles avanzados sobre autonomía del DT
- **IMPLEMENTATION_PLAN.md** - Plan de implementación (completado)
- **PROJECT_SUMMARY.md** - Resumen completo del proyecto
- **INDEX.md** - Índice antiguo de documentación
- **REORGANIZATION_PLAN.md** - Plan de reorganización de documentación

## Nota

Estos documentos son útiles para:
- Entender el contexto histórico del proyecto
- Ver la evolución de las decisiones de diseño
- Consultar referencias y fuentes de inspiración
- Entender planes y revisiones pasadas

**No son necesarios** para usar el framework en producción.
